## Bagian 1: Merespons Hasil Review dari Reviewer

### Tujuan Pembelajaran

Materi ini akan menunjukkan bagaimana merespon hasil telaah (review) dari reviewer yang kemudian harus dikomunikasikan kepada penulis untuk yang bersangkutan agar merevisi naskahnya.

### Langkah 1

Mulailah dengan mengidentifikasi naskah mana saja yang sudah direview dan sudah dikirim kepada kita selaku editor. Ditandai dengan keterangan **New reviews have been submitted**.

*(Gambar: daftar submission dengan status "New reviews have been submitted")*

Selanjutnya, cek terlebih dahulu hasil pekerjaan Reviewer.

*(Gambar: tampilan Review Files dan daftar Reviewers dengan status Review Submitted)*

Setelah Anda memeriksa pekerjaan Reviewer, berikutnya Anda perlu memberikan penilaian kepada Reviewer dan memberikan konfirmasi bahwa hasil Review sudah Anda terima dengan Klik **CONFIRM**.

*(Gambar: halaman Reviewer Comments — berisi contoh komentar "HARAP DIREVISI SESUAI CATATAN", daftar Reviewer Files, opsi Recommendation "Revisions Required", opsi Reviewer rating, dan tombol Confirm/Cancel)*

Yang terakhir Anda perlu menyampaikan ucapan terima kasih atas pekerjaan yang sudah dilakukan oleh Reviewer. Caranya dengan Klik tombol **Thank Reviewer**.

*(Gambar: daftar Reviewers dengan opsi Thank Reviewer, Revert, Decision; serta jendela pengiriman email ucapan terima kasih ke reviewer)*

Dengan demikian, Anda sudah berhasil mengirimkan ucapan terima kasih kepada Reviewer melalui Email yang bersangkutan.

Suatu hari nanti akan sangat dibutuhkan **(misalnya untuk kenaikan pangkat)** sebagai dokumen bukti bahwa yang bersangkutan sudah melaksanakan pekerjaan sebagai Reviewer Jurnal.

---

### Langkah 2

Pada Langkah 2, sebuah jendela akan terbuka yang memungkinkan Anda untuk mengunggah file kiriman Anda.

**Pertama**, Anda HARUS memilih Komponen Artikel. Ini memungkinkan sistem mengetahui apakah file tersebut adalah badan manuskrip, gambar, kumpulan data, dll. Ini harus dipilih sebelum file Anda akan diunggah.

*(Gambar: jendela Upload Submission File — tab 1. Upload File, dengan field Article Component dan area drag-and-drop)*

Setelah Anda membuat pilihan tersebut, Anda kemudian dapat mengupload file pertama Anda. Penting untuk diperhatikan bahwa Anda hanya dapat mengunggah satu file dalam satu waktu. File tambahan dapat diunggah nanti dalam prosesnya. Biasanya, file pertama ini akan menjadi badan naskah Anda. Tekan tombol Lanjutkan setelah file diunggah.

*(Gambar: tab 2. Review Details — menampilkan nama file yang sudah diunggah beserta tautan Edit)*

Setelah mengupload file, Anda akan diminta untuk mereview nama file tersebut. Gunakan tautan Edit untuk membuat perubahan apa pun.

Klik tombol **CONTINUE**.

Selanjutnya, Anda memiliki opsi untuk mengulangi proses untuk mengunggah file tambahan (mis., Kumpulan data atau gambar).

Setelah Anda selesai mengunggah semua file Anda, klik Selesai; ini akan menutup jendela unggah.

*(Gambar: tab 3. Confirm — menampilkan "File Added" dengan opsi Add Another File, Complete, Cancel)*

Anda akan dibawa kembali ke layar Kirim Artikel di mana Anda akan melihat file yang telah Anda unggah. Jika Anda perlu melakukan perubahan, luaskan panah biru di sebelah kiri file Anda dan lakukan perubahan menggunakan tautan Edit.

Klik 'Simpan dan Lanjutkan' untuk pindah ke **Langkah 3**.

---

### Langkah 3

Pada Langkah 3, Anda akan diminta untuk menambahkan lebih banyak informasi tentang kiriman, termasuk judul kiriman (dibagi menjadi awalan, judul, dan subjudul), dan abstrak. Jika jurnal memungkinkan pengiriman dalam berbagai bahasa, mengklik setiap bidang metadata akan menampilkan opsi untuk memasukkan metadata dalam bahasa lain yang diaktifkan, memungkinkan Anda untuk memasukkan judul, subtitle dan abstrak dalam bahasa lain. Gulir ke bawah…

*(Gambar: form Submit an Article, tahap 3. Enter Metadata — field Prefix, Title, Subtitle, Abstract)*

… Anda dapat menambahkan kontributor tambahan.

Anda dapat menambahkan lebih banyak kontributor (mis., Rekan penulis), dengan mengklik link Tambahkan Kontributor. Ini akan membuka jendela baru dengan bidang untuk memasukkan informasinya.

*(Gambar: jendela Add Contributor — field Name (First/Middle/Last Name), Contact (Email), Country; serta daftar List of Contributors)*

Tekan Simpan, dan kontributor baru akan muncul di layar.

*(Gambar: List of Contributors berisi dua nama dengan kolom Email, Role, Primary Contact, In Browse Lists)*

Anda mungkin juga melihat bidang tambahan untuk diisi, seperti kata kunci. Jika bahasa tambahan diaktifkan untuk jurnal, Anda dapat memasukkan metadata dalam bahasa ini. Mengklik kolom metadata akan menampilkan kolom untuk bahasa lain yang diaktifkan di jurnal.

*(Gambar: field Additional Refinements — Keywords dengan contoh "elearning" dan "query management")*

Untuk memasukkan kata kunci, cukup ketik kata atau frase dan tekan tombol Enter Anda. Kata atau frase tersebut akan diformat sebagai kata kunci.

Klik Simpan dan Lanjutkan untuk maju ke **langkah 4**.

---

### Langkah 4

Pada Langkah 4, Anda akan diminta untuk mengonfirmasi bahwa Anda sudah yakin dengan kiriman Anda.

Klik **Selesaikan Pengiriman**.

*(Gambar: tahap 4. Confirmation — tombol Finish Submission dan Cancel)*

Sebuah kotak akan muncul meminta Anda untuk mengonfirmasi bahwa Anda sudah selesai. Klik OK.

*(Gambar: kotak konfirmasi "Are you sure you wish to submit this article to the journal?" dengan tombol OK/Cancel)*

Pengiriman Anda sekarang sudah selesai!

Editor telah diberitahu tentang kiriman Anda. Pada tahap ini, Anda dapat mengikuti tautan ke:

1. Tinjau kiriman ini
2. Buat kiriman baru
3. Kembali ke dasbor Anda

Setelah Anda menyelesaikan pengiriman, Anda tidak dapat mengubahnya. Jika Anda ingin mengganti file yang Anda kirimkan atau membuat perubahan lain pada kiriman, Anda perlu menghubungi editor melalui alat Diskusi Pra-Tinjauan.

*(Gambar: halaman "Submission complete" — tautan Review this submission, Create a new submission, Return to your dashboard)*

## Bagian 2: Mengomunikasikan Hasil Review kepada Penulis

### Tujuan Pembelajaran

Materi ini akan menunjukkan bagaimana mengkomunikasikan hasil review dari reviewer yang kemudian disampaikan kepada penulis supaya yang bersangkutan merevisi naskahnya.

### Langkah 1: Memastikan Identitas Reviewer Tidak Muncul di Tampilan MS Word

Dalam proses review naskah yang bersifat **Single Blind** atau **Double Blind**, Identitas Reviewer tidak boleh diketahui oleh Penulis. Maka sebelum naskah dikirim ke penulis untuk direvisi, tugas Editor adalah memastikan Identitas Reviewer aman atau tidak muncul di halaman MS Word.

*(Gambar: contoh tampilan MS Word dengan komentar bernama "Jamileh" yang berpotensi membocorkan identitas reviewer)*

Bagaimana cara untuk mengganti nama komentar agar identitas sebagai reviewer tidak diketahui oleh penulis?

Pada percobaan ini digunakan Word Office 365.

- Pilih menu **File** kemudian pilih **Options**.
- Pada menu **General** bagian **Personalize your copy of Microsoft Office**.
- Centang pada kalimat **Always use these values regardless of sign in to Office** dan **User name** silahkan diketikkan apa saja, identitas Anda saat ini sesuai dengan apa yang diisi pada User name ini.

*(Gambar: dialog Personalize your copy of Microsoft Office — field User name diisi "REVIEWER A", checkbox "Always use these values regardless of sign in to Office" dicentang)*

*(Gambar: hasil komentar setelah nama diganti, identitas asli sudah tidak terlihat)*

Selanjutnya Anda perlu mengunggah perubahan file hasil menyembunyikan identitas reviewer (jika diperlukan) pada menu upload File dengan klik **Review Details** terlebih dahulu.

*(Gambar: panel Reviewers dengan tautan Review Details, Email Reviewer, Edit, Cancel Reviewer, History, Login As, Editorial Notes; serta daftar Reviewer Files termasuk file "EDITED CONTOH HASIL REVIEW")*

---

### Langkah 2: Mengirim Pesan Revisi Kepada Penulis

1. Klik tombol **Request Revisions**
2. Centang file MS Word yang akan dikirim ke penulis, kemudian Klik **Record Editorial Decisions**

*(Gambar: tab Review pada Workflow — Round 1 Status "New reviews have been submitted", tombol Request Revisions/Accept Submission/Decline Submission)*

*(Gambar: jendela Request Revisions — opsi Require New Review Round, opsi Send Email ke penulis dengan draf pesan keputusan "Revisions Required", daftar Send to Reviewers, dan daftar file review yang dapat dibagikan ke penulis (Select review files to share with the author(s)))*

## Bagian 3: Menanggapi Kiriman Hasil Revisi dari Penulis

### Tujuan Pembelajaran

Materi ini akan memungkinkan Anda mempelajari tentang langkah-langkah yang diambil editor dalam menanggapi kiriman hasil Revisi dari Penulis.

1. Cek hasil revisi dari penulis. Jika sudah dirasa tidak ada masalah, selanjutnya Anda harus memberikan keputusan dengan klik **Accept Submission**

*(Gambar: panel Review Files, Reviewers, dan Revisions — menampilkan file revisi "CONTOH REVISI ARTIKEL" yang diunggah penulis, serta tombol Request Revisions/Accept Submission/Decline Submission)*

## Bagian 4: Mengelola Tahap Copyediting

### Tujuan Pembelajaran

Materi ini akan menunjukkan kepada Anda bagaimana mengelola tahap penyiapan naskah pasca review, yaitu pengecekan tata bahasa, tanda baca, dan layout, termasuk bekerja dengan penulis dan penyunting.

### Langkah 1: Menunjuk Petugas Copy Editor

Ketika kiriman diterima di Tahap Review, itu akan secara otomatis pindah ke tahap Copyediting. Ketika pengajuan memasuki Tahap Copyediting, pemberitahuan menunjukkan bahwa Copyeditor perlu ditugaskan. Copyeditors dapat ditugaskan menggunakan tautan **Add** link pada Participants panel.

*(Gambar: tab Copyediting — Notification "Assign a copyeditor using the Assign link in the Participants list", daftar Draft Files, Copyediting Discussions, dan Copyedited)*

Ini akan membuka jendela baru.

- Pilih salah satu Copyeditors untuk kiriman ini.
- Pilih pesan yang sudah disiapkan.
- Tekan **Send**.

*(Gambar: jendela Assign Participant — dropdown Copyeditor, search user by name, daftar nama kandidat (La Hewi, Irmawati, Denok Dwi Anggraini))*

Maka akan terjadi perubahan pada bagian Participant (orang-orang yang terlibat menangani naskah).

*(Gambar: panel Participants menampilkan Journal editor, Copyeditor (La Hewi), dan Author)*

Tugas seorang Copy editor adalah melakukan pengecekan tata letak, Typografi, tata bahasa, tanda baca, dan layout, termasuk memeriksa judul artikel, abstrak, nama kontributor, kata kunci, dll.

Berikut halaman antar muka saat login pada akun Copy Editor.

*(Gambar: tampilan workflow Submission/Review/Copyediting/Production dari sudut pandang akun Copy Editor, dan tab Publication dengan menu Title & Abstract, Contributors, Metadata, References, Identifiers, Galleys, Permissions & Disclosure, Issue)*

---

### Langkah 2: Mendiskusikan dengan Penulis

Setelah selesai pemeriksaan oleh Copyeditor, selanjutnya akan dikonsultasikan kepada penulis dengan menambahkan diskusi baru, menggunakan tautan **Copyediting Discussions ==> Add discussion**.

*(Gambar: panel Copyediting Discussions dengan tombol Add discussion)*

Dari jendela Tambah Diskusi, copyeditor memilih Penulis dan menambahkan baris subjek dan pesan.

*(Gambar: jendela Add discussion — daftar Participants (La Hewi/Copyeditor, Jamiludin Usman/Journal editor, Author), field Subject "Cek naskah sebelum dipublish", field Message berisi contoh permintaan "Mohon kesediaannya untuk mengecek kebenaran metadata penulis (nama, asal institusi, alamat email, dll)", Attached Files, dan tombol OK/Cancel)*

Selanjutnya COPYEDITOR menunggu tanggapan penulis untuk memeriksa naskahnya.

*(Gambar: daftar Copyediting Discussions menampilkan diskusi "Cek naskah sebelum dipublish" dari La Hewi tertanggal 2024-12-13)*

---

### Langkah 3: Final Copyediting

Setelah berhasil menyelesaikan diskusi bersama penulis, hasil akhir naskah yang siap dipublis diupload pada bagian **Copyedited**.

Kemudian langkah berikutnya adalah mengirim pada tahapan produksi.

Klik **Send To Production**.

*(Gambar: panel Copyediting Discussions dan Copyedited — file "CONTOH COPYEDITING ARTIKEL" yang sudah diunggah, tombol Send To Production)*

*(Gambar: jendela Send To Production — daftar Draft Files dan Copyedited yang akan diteruskan ke tahap Production, tombol Record Editorial Decision, Previous: Author Notification, Cancel)*

Dengan demikian, Pekerjaan Copyeditor sekarang sudah selesai!

## Bagian 5: Template Artikel dan Prinsip Layout

### A. Prinsip Dasar Layout Jurnal

Mengacu pada *Principles of Transparency and Best Practice in Scholarly Publishing*.

| No | Prinsip | English | Indonesian |
|----|---------|---------|------------|
| 1 | Website | A journal's website, including the text that it contains, shall demonstrate that care has been taken to ensure high ethical and professional standards. It must not contain information that might mislead readers or authors, including any attempt to mimic another journal/publisher's site. An 'Aims & Scope' statement should be included on the website and the readership clearly defined. There should be a statement on what a journal will consider for publication including authorship criteria (e.g. not considering multiple submissions, redundant publications). ISSNs should be clearly displayed (separate for print and electronic). | Situs web jurnal, termasuk teks yang dikandungnya, harus menunjukkan bahwa kehati-hatian telah dilakukan untuk memastikan standar etika dan profesional yang tinggi. Tidak boleh mengandung informasi yang dapat menyesatkan pembaca atau penulis, termasuk segala upaya untuk meniru situs jurnal/penerbit lain. Focus & Scope harus disebutkan dengan jelas di situs web dan jumlah pembaca harus didefinisikan dengan jelas. Harus ada pernyataan tentang apa yang akan dipertimbangkan jurnal untuk publikasi termasuk kriteria kepengarangan (misalnya, tidak boleh submit di beberapa tempat, publikasi yang berlebihan). ISSN harus ditampilkan dengan jelas (terpisah untuk cetak dan elektronik). |
| 2 | Journal name | The Journal name shall be unique and not be one that is easily confused with another journal or that might mislead potential authors and readers about the Journal's origin or association with other journals. | Nama Jurnal harus unik dan tidak mudah tertukar dengan jurnal lain atau yang dapat menyesatkan calon penulis dan pembaca tentang asal-usul Jurnal atau keterkaitannya dengan jurnal lain. |
| 3 | Peer review process | Journal content must be clearly marked as whether peer reviewed or not. Peer review is defined as obtaining advice on individual manuscripts from reviewers' expert in the field who are not part of the journal's editorial staff. This process, as well as any policies related to the journal's peer review procedures, shall be clearly described on the journal's website, including the method of peer review used. Journal websites should not guarantee manuscript acceptance or very short peer review times. | Konten jurnal harus dinyatakan dengan jelas apakah melalui peer review atau tidak. Peer review adalah catatan saran tentang manuskrip seseorang dari pengulas ahli di bidang tertentu yang bukan bagian dari staf editorial jurnal. Proses ini, serta kebijakan apa pun yang terkait dengan prosedur peer review jurnal, harus dijelaskan secara jelas di situs web jurnal, termasuk metode peer review yang digunakan. Situs web jurnal tidak boleh menjamin penerimaan naskah atau waktu tinjauan sejawat yang sangat singkat. |
| 4 | Ownership and management | Information about the ownership and/or management of a journal shall be clearly indicated on the journal's website. Publishers shall not use organizational or journal names that would mislead potential authors and editors about the nature of the journal's owner. | Informasi tentang kepemilikan dan/atau pengelolaan jurnal harus dicantumkan dengan jelas di situs web jurnal. Penerbit tidak boleh menggunakan nama organisasi atau jurnal yang akan menyesatkan calon penulis dan editor tentang sifat pemilik jurnal. |
| 5 | Governing body | Journals shall have editorial boards or other governing bodies whose members are recognized experts in the subject areas included within the journal's scope. The full names and affiliations of the journal's editorial board or other governing body shall be provided on the journal's website. | Jurnal harus memiliki dewan redaksi atau badan pengatur lainnya yang anggotanya diakui sebagai ahli dalam bidang studi yang termasuk dalam ruang lingkup jurnal. Nama lengkap dan afiliasi dewan redaksi jurnal atau badan pengatur lainnya harus disediakan di situs web jurnal. |
| 6 | Editorial team/contact information | Journals shall provide the full names and affiliations of the journal's editors on the journal's website as well as contact information for the editorial office, including a full address. | Jurnal harus mencantumkan nama lengkap dan afiliasi editor jurnal di situs web jurnal serta informasi kontak untuk kantor redaksi, termasuk alamat lengkap. |
| 7 | Copyright and Licensing | The policy for copyright shall be clearly stated in the author guidelines and the copyright holder named on all published articles. Likewise, licensing information shall be clearly described in guidelines on the website, and licensing terms shall be indicated on all published articles, both HTML and PDFs. If authors are allowed to publish under a Creative Commons license, then any specific license requirements shall be noted. Any policies on posting of final accepted versions or published articles on third party repositories shall be clearly stated. | Kebijakan hak cipta harus dinyatakan dengan jelas dalam pedoman penulis dan nama pemegang hak cipta pada semua artikel yang diterbitkan. Demikian juga, informasi lisensi harus dijelaskan dengan jelas dalam pedoman di situs web, dan persyaratan lisensi harus ditunjukkan pada semua artikel yang diterbitkan, baik HTML maupun PDF. Jika penulis diizinkan untuk menerbitkan di bawah lisensi Creative Commons, maka persyaratan lisensi tertentu harus dicatat. Setiap kebijakan tentang pengeposan versi final yang diterima atau artikel yang diterbitkan pada repositori pihak ketiga harus dinyatakan dengan jelas. |
| 8 | Author fees | Any fees or charges that are required for manuscript processing and/or publishing materials in the journal shall be clearly stated in a place that is easy for potential authors to find prior to submitting their manuscripts for review, or explained to authors before they begin preparing their manuscript for submission. If no such fees are charged that should also be clearly stated. | Setiap biaya atau pungutan yang diperlukan untuk pemrosesan naskah dan/atau materi penerbitan dalam jurnal harus dinyatakan dengan jelas di tempat yang mudah ditemukan oleh calon penulis sebelum menyerahkan naskah mereka untuk ditinjau atau dijelaskan kepada penulis sebelum mereka mulai menyiapkan naskah mereka untuk penyerahan. Jika tidak ada biaya yang dibebankan, itu juga harus dinyatakan dengan jelas. |
| 9 | Process for identification of and dealing with allegations of research misconduct | Publishers and editors shall take reasonable steps to identify and prevent the publication of papers where research misconduct has occurred, including plagiarism, citation manipulation, and data falsification/fabrication, among others. In no case shall a journal or its editors encourage such misconduct, or knowingly allow such misconduct to take place. In the event that a journal's publisher or editors are made aware of any allegation of research misconduct relating to a published article in their journal, the publisher or editor shall follow COPE's guidelines (or equivalent) in dealing with allegations. | Penerbit dan editor harus mengambil langkah-langkah yang wajar untuk mengidentifikasi dan mencegah publikasi makalah di mana kesalahan penelitian telah terjadi, termasuk plagiarisme, manipulasi kutipan, dan pemalsuan/fabrikasi data, antara lain. Dalam hal apa pun jurnal atau editornya tidak boleh mendorong perilaku yang salah seperti itu, atau dengan sengaja membiarkan pelanggaran tersebut terjadi. Dalam hal penerbit atau editor jurnal mengetahui adanya dugaan pelanggaran penelitian yang berkaitan dengan artikel yang diterbitkan dalam jurnal mereka, penerbit atau editor harus mengikuti pedoman COPE (atau yang setara) dalam menangani tuduhan. |
| 10 | Publication ethics | A journal shall also have policies on publishing ethics. These should be clearly visible on its website, and should refer to: i) Journal policies on authorship and contributorship; ii) How the journal will handle complaints and appeals; iii) Journal policies on conflicts of interest / competing interests; iv) Journal policies on data sharing and reproducibility; v) Journal's policy on ethical oversight; vi) Journal's policy on intellectual property; and vii) Journal's options for post-publication discussions and corrections. | Jurnal juga harus memiliki kebijakan etika publikasi. Ini harus terlihat jelas di situs webnya, dan harus mengacu pada: i) Kebijakan jurnal tentang kepengarangan dan kontributor; ii) Bagaimana jurnal akan menangani keluhan dan banding; iii) Kebijakan Jurnal tentang benturan kepentingan/kepentingan yang bersaing; iv) Kebijakan jurnal tentang berbagi data dan reproduktifitas; v) kebijakan Jurnal tentang pengawasan etis; vi) kebijakan Jurnal tentang kekayaan intelektual; dan vii) Pilihan jurnal untuk diskusi dan koreksi pasca-publikasi. |
| 11 | Publishing schedule | The periodicity at which a journal publishes shall be clearly indicated. | Periode penerbitan jurnal harus ditunjukkan dengan jelas. |
| 12 | Access | The way(s) in which the journal and individual articles are available to readers and whether there is associated subscription or pay per view fees shall be stated. | Cara jurnal dan artikel individual tersedia untuk pembaca dan apakah ada biaya berlangganan atau bayar per tampilan terkait harus disebutkan. |
| 13 | Archiving | A journal's plan for electronic backup and preservation of access to the journal content (for example, access to main articles via CLOCKSS or PubMedCentral) in the event a journal is no longer published shall be clearly indicated. | Rencana jurnal untuk pencadangan elektronik dan pelestarian akses ke konten jurnal (misalnya, akses ke artikel utama melalui CLOCKSS atau PubMedCentral) jika jurnal tidak lagi diterbitkan harus ditunjukkan dengan jelas. |
| 14 | Revenue sources | Business models or revenue sources (eg, author fees, subscriptions, advertising, reprints, institutional support, and organizational support) shall be clearly stated or otherwise evident on the journal's website. Publishing fees or waiver status should not influence editorial decision making. | Model bisnis atau sumber pendapatan (misalnya, biaya penulis, langganan, iklan, cetak ulang, dukungan institusional, dan dukungan organisasi) harus dinyatakan dengan jelas atau terbukti di situs web jurnal. Biaya penerbitan atau status pengabaian tidak boleh mempengaruhi pengambilan keputusan editorial. |
| 15 | Advertising | Journals shall state their advertising policy if relevant, including what types of adverts will be considered, who makes decisions regarding accepting adverts and whether they are linked to content or reader behavior (online only) or are displayed at random. Advertisements should not be related in any way to editorial decision making and shall be kept separate from the published content. | Jurnal harus menyatakan kebijakan periklanannya jika relevan, termasuk jenis iklan apa yang akan dipertimbangkan, siapa yang membuat keputusan terkait penerimaan iklan dan apakah iklan tersebut terkait dengan konten atau perilaku pembaca (hanya online) atau ditampilkan secara acak. Iklan tidak boleh terkait dengan cara apa pun dengan pengambilan keputusan editorial dan harus disimpan terpisah dari konten yang dipublikasikan. |
| 16 | Direct marketing | Any direct marketing activities, including solicitation of manuscripts that are conducted on behalf of the journal, shall be appropriate, well targeted, and unobtrusive. Information provided about the publisher or journal is expected to be truthful and not misleading for readers or authors. | Setiap kegiatan pemasaran langsung, termasuk permintaan manuskrip yang dilakukan atas nama jurnal, harus sesuai, tepat sasaran, dan tidak mengganggu. Informasi yang diberikan tentang penerbit atau jurnal diharapkan jujur dan tidak menyesatkan bagi pembaca atau penulis. |

#### 2. Dasar Perlunya Layout/Grafis

**2.6 Penampilan**

Jurnal ilmiah disajikan dengan tampilan format secara konsisten, harmonis, dan berciri khas. Format yang dimaksud meliputi bentuk, ukuran bidang tulisan, lebar pinggir bidang tulisan, jarak antarkalimat, dan pemilihan jenis huruf. Enam kriteria penilaian penampilan mengikuti ketentuan berikut:

1. **Ukuran Bidang Tulisan.** Ukuran bidang tulisan jurnal ilmiah harus sesuai dengan *style sheet* yang dijadikan pegangan oleh penyunting pelaksana dalam memapankan gaya selingkung jurnalnya. Ukuran bidang tulisan dari jurnal sebaiknya mengikuti standar UNESCO, yaitu A4 (210 mm × 297 mm).
2. **Tata Letak.** Tata letak (*layout*) mencakup penataan ruang halaman, penempatan baris judul, paragraf, dan ilustrasi. Tata letak di setiap terbitan harus konsisten karena menentukan tampilan halaman artikel dan mencirikan gaya selingkung jurnal ilmiah. Tata letak yang terlalu banyak menampilkan ruang kosong sebaiknya dihindari.
3. **Tipografi.** Konsistensi tipografi meliputi pilihan jenis, bentuk, dan ukuran huruf, penyetelan jarak antarbaris, jarak antarhuruf (*kerning*), perataan tepi bidang tulisan, dan ragamnya.
4. **Resolusi Dokumen.** Dokumen dalam format portabel (*portable document format*, PDF) memiliki mutu tulisan yang perlu dijaga konsistensinya dan beresolusi tinggi.
5. **Jumlah Halaman per Volume (Jilid).** Setiap volume paling sedikit berjumlah 100 halaman. Satu volume tidak harus diselesaikan dalam 1 tahun kalender. Penilaian jumlah halaman per terbitan dimaksudkan untuk memberi nilai lebih tinggi kepada jurnal yang dapat menerbitkan artikel dalam jumlah banyak tetapi dengan mutu yang tetap terjaga baik.
6. **Desain Tampilan Laman (*Website*) dan Desain Sampul.** Tampilan laman dan desain sampul memiliki kekhasan. Informasi penting seperti tim penyunting, petunjuk penulisan, tujuan penerbitan, bidang ilmu yang dilingkupi, nama penerbit, dan alamat jurnal ditampilkan pada halaman depan. Hal ini untuk memberikan informasi yang jelas dan kemudahan bagi pengguna. Etika publikasi dan daftar pengindeks juga sebaiknya ditambahkan di menu utama halaman depan laman.

*Sumber: Direktorat Jenderal Penguatan Riset dan Pengembangan Kementerian Riset, Teknologi, dan Pendidikan Tinggi, Pedoman Akreditasi Jurnal Ilmiah (Jakarta: Kemenristekdikti, 2018), 19-21.*
